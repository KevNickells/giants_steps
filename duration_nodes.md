Duration per chord (seconds) = 1200 ÷ N

16 chords   1200 ÷ 16 = 75s / chord     once
32 chords   1200 ÷ 32 = 37.5s / chord   twice
48 chords   1200 ÷ 48 = 25s / chord     triply
64 chords   1200 ÷ 64 = 18.75s / chord  quatricely
80 chords   1200 ÷ 80 = 15s / chord     quinticely
96 chords   1200 ÷ 96 = 12.5s / chord   sexticely
